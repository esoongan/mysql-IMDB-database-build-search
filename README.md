## 📌  프로젝트 소개
대용량의 imdb 데이터셋을 이용하여 Relationao Database를 구축하고 검색하는 프로그램입니다.
각 파일마다 최소 천만개~억개이상의 데이터를 가진 대용량 데이터를 효율적으로 검색하기위하여 E-R 모델링, 테이블재설계, 인덱스를 활용하였습니다.
